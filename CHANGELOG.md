# Changelog

## [0.1.0] - 2026-03-05

### Added
- VSCode extension with commands:
  - `refactor.analyze` - Analyze code for smells
  - `refactor.rewrite` - AI-powered code rewriting
  - `refactor.visualize` - Dependency graph visualization
- Core features:
  - Smell detection (20+ types)
  - Complexity calculation
  - Constraint extraction (explicit + implicit)
  - Dependency graph building
- Integrations:
  - Codex/Gemini AI rewriter
  - Functional equivalence verifier
  - RefakTS precise refactoring
  - EvoMap self-evolving
- WebView UI with D3.js visualization
